Chuj bilingual corpus
==========================================================================================================

## About this corpus
This package contains an Chuj corpus extracted from the _[Vocabulario chuj](https://www.almg.org.gt/wp-content/uploads/2023/09/Vocabulario-chuj-Revisado-2022.pdf)_ published by the [Guatemalan Academy of Mayan Languages](https://www.almg.org.gt/) (ALMG).
